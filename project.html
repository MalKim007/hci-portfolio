Beach Play: Learn & Play by the Sea!


Beach Play is a free educational app that makes learning English and math fun for kids aged 7-12! Through interactive games and a vibrant beach theme, kids can improve their vocabulary, grammar, pronunciation, and math skills (addition, subtraction, multiplication, division).

Key Features:

  • Free to download and use
  • Two Subjects: English and Math
  • Four Math Operations: Addition, Subtraction, Multiplication, Division
  • Engaging Beach Theme: Fun visuals and a calming environment
  • Interactive Gameplay: Keeps kids focused and entertained
  • Ages 7-12: Designed for elementary school children

Beach Play offers a fun alternative to traditional learning, helping kids master basic math and improve their English in an engaging way. It's a more beneficial use of screen time than typical games.

Download Beach Play today and let the learning fun begin!

GetSports: Sports Equipment Booking System


GetSports is a Progressive Web App (PWA) designed to streamline sports equipment booking for students and staff at Politeknik Sultan Idris Shah (PSIS). It provides a convenient way to check equipment availability, make bookings, and stay informed about sports news and events.

Project Goals:

  • Improve management and accessibility of sports equipment.
  • Provide a centralized platform for sports-related information.
  • Increase participation in sports activities.
  • Contribute to a more active and healthy campus community.

Development:

Developed using the Rapid Application Development (RAD) model, GetSports was built to be fast and cost-effective. Future improvements include a notification system for booking reminders and equipment maintenance, as well as ongoing user testing and feedback integration.


GetSports has the potential to significantly enhance the sporting experience at PSIS by providing a user-friendly and efficient equipment booking solution.

myJPJ Web Services



During my internship, I contributed to the development of a SOAP-based API designed to retrieve public user data. This API provides a standardized way for authorized systems to access and query user information based on various identifiers. The project involved creating a robust and secure service that adheres to SOAP standards and integrates seamlessly with existing systems.

Key Features:

  • SOAP Protocol: The API is built using the SOAP protocol, ensuring standardized communication and data exchange with consuming applications.
  • JSON Responses: Although it's a SOAP API, it conveniently returns data in JSON format, making it easier for modern applications to process the responses.

Technologies:

  • Laravel (PHP): The API was developed using the Laravel framework in PHP, providing a structured and efficient way to build web services.
  • SOAP (Simple Object Access Protocol): Used for message exchange between the API and its clients.
  • SoapClient (PHP): The built-in PHP SoapClient class was used to handle SOAP requests and responses.
  • Postman: Used for testing and documenting the API endpoints.
